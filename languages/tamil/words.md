# குறில் நெடில்

```
படு வட்டம் நடு நிலம் அடை கல் மடு தனம் அவல் கடை 
மலை குறை துக்கம் திட்டு கொடு சிலை தொடு சுடு முடி கொடி 
படம் பனை மடம் வலி வடை குடம் பல் நகம் பதம் தடி
பட்டு மடி குரை சட்டை புட்டு கட்சி விடு மென்மை படி வடு 
வெட்டு கொடு உரல் தடை முட்டை மெய் குடை எடு அடி கெட்டு
```

# வார்த்தை அமை

```
சேவல் கத்தி தேள் ஔவையார் ஊஞ்சல்  எறும்பு எஃகு சீப்பு நண்டு ஒட்டகம் 
ஐவர் அன்னம் பந்து உண்டியல் தாழ்ப்பாள் சங்கு செவ்வாழை நாற்று சக்கரம் ஓடம் 
பட்டம் வட்டம் நிலம் தனம் துக்கம் திட்டு படம் மடம் குடம் பட்டு 
சட்டை புட்டு கட்சி மென்மை வெட்டு உரல் முட்டை பாட்டு சாட்டை  பூட்டு  
காட்சி மேன்மை வேட்டு மூட்டை  வாட்டம்   தானம் ஆவல் தூக்கம் பாடம் மாடம் 
நாகம் பாதம் 
```

# தொழில் (வேலை)

```
ஆடை நெய்பவர்  - நெசவாளர் 
உழவுத் தொழில் செய்பவர்  - உழவர் 
வான் ஊர்தி ஓட்டுபவர்  - விமானி 
சிலை வடிப்பவர்  - சிற்பி 
ஓவியம் வரைபவர்  - ஓவியர் 
தூய்மைப்பணி செய்பவர்  - துப்புரவாளர் 
பானைகள் செய்பவர்   - குயவர் 
யானையைப் பழக்குபவர்  - பாகன் 
பொன்வேலை செய்பவர்  - பொற்கொல்லர் 
மரவேலை செய்பவர்  - தச்சர் 
பொருள்களை விற்பவர்  - வணிகர் 
கப்பலைச் செலுத்துபவர்  - மாலுமி 
ஊர்திகளை ஓட்டுபவர்  - ஓட்டுநர் 
மீன் பிடிப்பவர்  - மீனவர் 
வேட்டையாடுபவர்  - வேடர் 
நாடகங்களில் நடிப்பவர்  - நடிகர் 
கதை, கட்டுரை எழுதுபவர்  - எழுத்தாளர்   
வழக்குகளை எடுத்துரைப்பவர்  - வழக்குரைஞர்     
நீதி வழங்குபவர்  - நீதி அரசர் 
கடிதங்களை சேர்ப்பவர்  - அஞ்சலகர் 
மருத்துவம் பார்ப்பவர்  - மருத்துவர் 
பாடல்களை பாடுபவர்  - பாடகர்  
```
