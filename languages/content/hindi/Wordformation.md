
# Word Formation (शब्द रचना)

# 3 Letters word
```

क + म + ल = कमल 
म + ग + र = मगर
ब + त +ख = बतख 
औ + र + त = औरत
अ + क्ष + र = अक्षर
बा + न + र = बानर
```

```
फ + स + ल = फसल
म + ह + ल = महल
ट + ह + ल = टहल
ख + ब + र = खबर
न + ग + र = नगर
अ  + ग + र = अगर
```

```
क + ल +श = कलश
प + ल +क = पलक
भ + ग + त = भगत
त + ख + त = तखत
श + ह + द = शहद
प + व + न = पवन
```

```
भ + ज + न = भजन
ज + ल + न = जलन
न + म + न = नमन
द  + म + न = दमन
च + म + क = चमक
ल + प + क = लपक
```

```
प + ट + क = पटक
भ + व + न = भवन
ह + व + न = हवन
क + न + क = कनक
च + म + न = चमन
न + य + न = नयन
```
