
# Homonyms ( श्रुतिसमभिन्नार्थक शब्द )

# Reading 

जो शब्द सुनने और उच्चारण करने में समान प्रतीत हों, किन्तु उनके अर्थ भिन्न -भिन्न हों, वे श्रुतिसमभिन्नार्थक / समोच्चरित शब्द कहलाते हैं ।

```

दरवाज़ा - *कपाट (कपट)*
 
दिवस - *दिन (दीन)*

```

```

```
