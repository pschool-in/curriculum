```

```

```
शहर में रहने वाला - *शहरी* 
जो विज्ञान जनता है - *वैज्ञानिक*
जहाँ इलाज किया जाता हो - *अस्पताल*

जो जूते ठीक करता है - *मोची* 
जो रोग से ग्रस्त है - *रोगी* 
```

```

```
