# Hindi Sentence (वाक्य)

# Level 1
```

धीरे बोलो।
तेज़ बोलो।
बारिश हो रही है।
सुबह हो गयी।
मेरे साथ चलो।
उसे मत देखो।
तुम सीख जाओगे।

```

```
जूते पहन लो।
बहाने मत बनाओ।
उसे समय लेने दो।
इशारा मत करो।
ज़िद मत करो।
अंधेरा हो रहा है।
मुझे नहाने दो।
जाकर सो जाओ।
```

# Level 3

```
मुझे तैरना आता है।
मैं तुमसे प्यार करता हूँ।
मैं कोशिश करूँगा।
मैं आ रहा हूँ।
मुझे भूख लगी है! 
जन्मदिन मुबारक हो!
मुझे गुस्सा मत दिलाओ।
मुझे ठंड लग रही है।
```
```
नंगे पैर मत चलो।
दोबारा कब मिलेंगे?
इंडिया जीतेगी।
तुम अंग्रेज़ी सीखोगे।
ढक्कन खोल दो।
अपने बाल बना लो।

```

# Level 4
```

उसे पैसों की ज़रूरत है।
मुझे इतिहास पसंद है।
मुझे यह कुत्ता अच्छा लगता है।
मैं वापस आऊँगा।
मैं घर पर ही रहूँगा।
मैं बहुत थक गया हूँ। 
आज मौसम बहुत गरम है।
```

```
बिस्तर पर चादर बिछा दो।
दीवार पर सहारा मत लो।
दिमाग से काम लो।
समझने की कोशिश करो।
वह भागते हुए आया।
वह बीमार नहीं हो सकता।
उसने टोपी पहनी हुई है। 
```

# Level 5
```

सब लोग कैसे हैं?
तुम कितने साल के हो? 
मुझे उल्टी आ रही है।
मैं हर साल जाता हूँ।
मुझे कोई आवाज़ सुनाई दी। 
मैं रोज़ दौड़ने जाता हूँ।
मैंने उसे फिरसे देखा। 
```

```
मुझे सच्चाई बताओ।
एक बार फिरसे कोशिश करो।
हम उसे नहीं जानते हैं।
हम जंग के ख़िलाफ़ हैं।
हम युद्ध का विरोध करते हैं। 
खुशी क्या होती है?
आपका नाम क्या है?
```

# एकवचन और बहुवचन (drag and drop)

उचित शब्दों से रिक्त स्थानों की पूर्ति कीजिए |

```

मेरे *भाइयों* ने मिलकर मुझे तोहफा दिया है ।
मेरा *भाई* कहा चला गया। 
मंत्री जी के सारे *बेटियाँ*  आए थे।
मंत्री जी की *बेटी* आयी थी। 
बहुत सारे *व्यंजन* बने हैं। 
इन सब *व्यंजनों* में चीनी है।
```
```
वह *लड़का* उन *लड़कों* से पिटा है।
मेरी सारी *चिट्ठियाँ* लौटा दो।
अमन के लिए *चिट्ठी*आयी है। 
संगम पर तीन *नदियाँ* मिलती हैं।
बच्चे *नदी*  किनारे कल रहे थे। 
```
```
सभी *पेड़* सेबों से लदे हुए हैं पर इन दो* पेड़ों* पर कोई फल नहीं है।
मैंने एक *संतरा* खाया।
उसने दो *संतरे* खाये।
*लड़की* खेलती है।
*लड़कियाँ* खेलती हैं।
```
```
*कुत्ता* भौंक रहा है।
*कुत्ते* भौंक रहे हैं।
लंगूरों की एक *टोली* ने बहुत उत्पात मचा रखा है।
विद्यार्थियों की बहुत सी *टोलियाँ* गई हैं।
```
