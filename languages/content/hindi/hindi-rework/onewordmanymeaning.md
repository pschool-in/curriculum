# Match the following

```
आम, साधारण और एक फल 
अंग, शरीर और शाखा
अंत, मृत्यु और भेद
उत्तर, जबाव और एक दिशा 
उपचार, सेवा और इलाज
```

```

गोल, आकृति और खेल में गोल
```
