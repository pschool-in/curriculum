
# Hindi Sentence (वाक्य)
```

धीरे बोलो।
तेज़ बोलो।
खाना लगा दो।
बहाने मत बनाओ।
कपड़े  सुखा दो।
धूप सेंक लो।
बिस्तर लगा दो।
ढक्कन खोल दो।
अपने बाल बना लो।
नल खोल दो।
बिस्तर पर चादर बिछा दो।
दीवार पर सहारा मत लो।
दिमाग से काम लो।
समझने की कोशिश करो।
```

```
जूते पहन लो।
मुझे गुस्सा मत दिलाओ।
उसे समय लेने दो।
इशारा मत करो।
बकवास मत करो।
ज़िद मत करो।
मुझसे पंगे मत लो।
अंधेरा हो रहा है।
मुझे नहाने दो।
जाकर सो जाओ।
मुझे ठंड लग रही है।
बारिश हो रही है।
फोन आ रहा है।
मशीन चालू करो।
```

```
एसी बन्द करो।
सुबह हो गयी।
मेरे साथ चलो।
उसे मत देखो।
ड्राइविंग सीख लो।
तुम सीख जाओगे।
नंगे पैर मत चलो।
दोबारा कब मिलेंगे?
इंडिया जीतेगी।
तुम अंग्रेज़ी सीखोगे।
तुम में कला है।
नौकरी कर लो।
उसे मत देखो।
उसे विदा कर दो।
```
