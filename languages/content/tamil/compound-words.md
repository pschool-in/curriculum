# பிரித்து எழுதுக
```
அச்செல்வம்=அ+செல்வம்
அந்நாடு=அ+நாடு
இப்பிணி=இ+பிணி
விலங்கினம்=விலங்கு+இனம்
வாழிடம்=வாழ்+இடம்
அரும்பணி=அருமை+பணி 
ஓரிடம் =ஓர் +இடம் 
பைங்குவளை =பசுமை+குவளை
மின்னஞ்சல் =மின்+அஞ்சல்
அமுதென்று=அமுது+என்று
```

```
நாட்குறிப்பு=நாள்+குறிப்பு
பன்மொழி=பல+மொழி
மூவாயிரம்=மூன்று+ஆயிரம்
பொற்காசு=பொன்+காசு
கற்கண்டு=கல்+கண்டு
அரசாணை=அரசு+ஆணை 
பேச்சுக்கலை=பேச்சு+கலை
மணமென்று=மணம்+என்று 
பேரிடர்=பெருமை+இடர்
பெரும்பகுதி=பெருமை+பகுதி
```

```
உடலுழைப்பு=உடல்+உழைப்பு
அறிவாற்றல்=அறிவு+ஆற்றல்
பதிவேடு=பதிவு+ஏடு
யாரவர்=யார்+அவர்
அறனல்ல=அறன்+அல்ல
படிப்பறிவு=படிப்பு+அறிவு
நீரமுது=நீர்+அமுது
என்றுரைத்தல்=என்று+உரைத்தல்
சண்டையில்லாமல்=சண்டை+இல்லாமல்
முடிவெடுத்தன=முடிவு+எடுத்தன
```
```
உணவகம்=உணவு+அகம் 
இடமின்றி= இடம்+இன்றி 
வந்திருந்த=வந்து+இருந்த
ஏனென்றால்=ஏன்+என்றால்
கண்ணுறங்கு=கண்+உறங்கு
ஒன்றையொன்று= ஒன்றை+ஒன்று
ஆளுக்கொரு=ஆளுக்கு+ஒரு
பணியாற்றும்=பணி +ஆற்றும்
செடியில்லாத=செடி+இல்லாத
மலர்த்தொட்டி =மலர்+தொட்டி 
```
```
மக்களுண்டு =மக்கள்+உண்டு
தமிழுண்டு=தமிழ்+உண்டு
புகைப்படம்= புகை+படம்
மலைத்தொடர்=மலை+தொடர் 
மலைக்குகை= மலை+குகை
கனியுண்டு=கனி+உண்டு 
கையொலி=கை +ஒலி
விளைந்திருந்தன = விளைந்து+இருந்தன 
நெடுநாள்=நெடு+நாள்
காலைச்சிற்றுண்டி=காலை+சிற்றுண்டி
```
```
எடுத்துவந்தார்=எடுத்து+வந்தார்
காட்டெருமை=காடு+எருமை 
பெருஞ்சண்டை=பெரும்+சண்டை 
கட்டவிழ்த்து=கட்டு+அவிழ்த்து 
கொள்வதேனோ=கொள்வது+ஏனோ  
நோக்கிப்போகவா=நோக்கி+போகவா 
கொள்ளலாயினன்=கொள்ளல்+ஆயினன் 
தொலைக்காட்சி=தொலை+காட்சி 
அறுவடைக்காலம்=அறுவடை+காலம் 
நச்சுப்பொருள்=நச்சு+பொருள்
```
```
பழச்சாறு = பழம்+சாறு
படக்கதை = படம்+கதை 
அறவழி=அறம்+வழி 
இரத்தமின்றி=  இரத்தம்+இன்றி
கதராடை = கதர்+ஆடை
தொண்டர்படை = தொண்டர்+படை 
பழமையான= பழமை+ஆன
செம்மொழி=செம்மை+மொழி 
கடுஞ்சொல்=கடுமை+சொல்
நூலாடை -நூல் +ஆடை
```
```
தானென்று=தான் +என்று
தன்னாடு=தன்+நாடு
பல்கலை=பல+கலை
பன்மொழி=பல+மொழி
பதிவேடு=பதிவு+ஏடு
கடுங்காவல்=கடுமை+காவல்
தளர்ந்திருந்த=தளர்ந்து+இருந்த
இன்னுயிர்=இனிமை+உயிர்
நல்லொழுக்கம்=நன்மை+ஒழுக்கம்
தோட்டம்+கலை =தோட்டக்கலை
```
```
காலமறிந்து=காலம்+அறிந்து
பற்பல=பல+பல
முத்தமிழ்=மூன்று+தமிழ்
நீராவி =நீர்+ஆவி 
புவியீர்ப்பு =புவி+ஈர்ப்பு 
தனக்கென்று =தனக்கு+என்று 
வீரக்கலை=வீரம்+கலை 
சிலம்பாட்டம் =சிலம்பு+ஆட்டம்
தீயொழுக்கம்=தீ+ஒழுக்கம்
மாடித்தோட்டம்=மாடி+தோட்டம்
```

```
என்றென்றும் =என்று+என்றும்
ஐம்பால் =ஐந்து+பால்
ஒலியெழுப்பி=ஒலி+எழுப்பி
பதவியேற்று=பதவி+ஏற்று
ஆற்றுநீர் =ஆறு+நீர்
வாழைப்பழம் =வாழை+பழம்  
மாட்டுவண்டி=மாடு+வண்டி
மரக்கிளை=மரம்+கிளை 
அன்புள்ளம் =அன்பு+உள்ளம்
பட்டாடை =பட்டு+ஆடை
```

```
நீலக்கடல் =நீலம்+கடல்
மரப்பலகை =மரம்+பலகை 
பாடநூல் =பாடம் +நூல் 
அச்சுக்கலை=அச்சு+கலை
தமிழெழுத்து=தமிழ்+எழுத்து
கண்ணெழுத்து= கண்+எழுத்து
அழகுணர்ச்சி=அழகு+உணர்ச்சி
ஓலைச்சுவடி=ஓலை+சுவடி
பனித்துளி=பனி+துளி
நல்லொழுக்கம்=நன்மை+ஒழுக்கம்
```

# சேர்த்து எழுதுக

```
கருமை + குயில்= கருங்குயில் 
படர்ந்து + இருக்கும்= படர்ந்திருக்கும்
வண்ணம் + மலர் = வண்ணமலர்
நெல் + கதிர் = நெற்கதிர் 
தேடி + சென்றன = தேடிச்சென்றன
பெருமை + ஒலி = பேரொளி
நுழைவு + சீட்டு =நுழைவுச்சீட்டு
தெரு + எங்கும்= தெருவெங்கும்
ஒலி + எழுப்பி =ஒலியெழுப்பி
இன்பம் + தமிழ்= இன்பத்தமிழ்
```
```
வேறு  + எங்கும் = வேறெங்கும்
வான்  + உயர்ந்த = வானுயர்ந்த 
நீர் + ஓசை =நீரோசை
மலையில் + இருந்து= மலையிலிருந்து
மானை + கண்டு =மானைக்கண்டு
நாடு + பற்று =நாட்டுப்பற்று
விடுதலை + போராட்டம் = விடுதலைப்போராட்டம்
செம்மை + மொழி = செம்மொழி
ஆறு + ஆயிரம் =ஆறாயிரம்
வடக்கு + மொழி=வடமொழி
```
```
தொழில்+கல்வி=தொழிற்கல்வி
இயற்கை+அன்னை=இயற்கையன்னை
நாடகம்+கல்வி=நாடகக்கல்வி
புதுமை+உயிர்=புத்துயிர்
நாடு+எல்லாம்=நாடெல்லாம்
சரிந்தது+அங்கே=சரிந்ததங்கே
வளர்ந்து+ஓங்கும்=வளர்ந்தோங்கும்
என்+உயிர்= என்னுயிர்
அறிவு+ஆயுதம்=அறிவாயுதம் 
அமர்ந்து + இருந்த=அமர்ந்திருந்த
```
```
கண்+இமைக்கும்=கண்ணிமைக்கும்
சூறை+காற்று =சூறைக்காற்று
மரம்+பொந்து=மரப்பொந்து
கருமை+கடல் =கருங்கடல் 
பழைமை+மொழி = பழமொழி
நன்மை+வழி = நல்வழி
செம்மை+தமிழ்=செந்தமிழ்
கவி+அரங்கம்=கவியரங்கம்
நன்மை+தமிழ்= நற்றமிழ் 
என்+உயிர்=என்னுயிர்
```
```
தெற்கு+மேற்கு=தென்மேற்கு
தமிழ்+சங்கம் =தமிழ்ச்சங்கம்
தொடக்கம்+விழா=தொடக்கவிழா
மயில்+ஆட்டம்=மயிலாட்டம்
குருவி+கூடு=குருவிக்கூடு
பூ+பந்தல் =பூப்பந்தல்
வேலை+ஆள் =வேலையாள் 
சமையல்+கலை=சமையற்கலை
அந்த+பெயர் =அந்தப்பெயர் 
இந்த+திருடன் =இந்தத்திருடன் 
```
```
எந்த+பக்கம்=எந்தப்பக்கம் 
குட்டி+பையன்=குட்டிப்பையன் 
தீ+பொறி=தீப்பொறி
நெசவு+தொழில் =நெசவுத்தொழில் 
விண் +உலகம்=விண்ணுலகம்
மரம்+கிளை =மரக்கிளை
வீதி+உலா =வீதியுலா 
வழி+எங்கும் =வழியெங்கும்    
மரம்+பெட்டி-மரப்பெட்டி 
கல்+குவியல்=கற்குவியல்
```
```
நீர்+கோவை=நீர்க்கோவை
உயிர்+தோழன் =உயிர்த்தோழன் 
நாடு+பற்று =நாட்டுப்பற்று 
வரகு+அரிசி=வரகரிசி 
வாக்கு +உரிமை=வாக்குரிமை
அமுது+என்று=அமுதென்று 
சூரியன்+வழிபாடு=சூரியவழிபாடு
வான்+ஒலி =வானொலி 
கால் +ஆண்டு =காலாண்டு
தமிழ் +சங்கம்=தமிழ்ச்சங்கம் 
```
```
மலை+கோட்டை =மலைக்கோட்டை 
வர+இல்லை =வரவில்லை 
எலி+பொறி =எலிப்பொறி 
முன்+காலம்==முற்காலம் 
காலம் +சக்கரம்=காலச்சக்கரம்
உறுதி+உடன் =உறுதியுடன்
அளவு+இல்லாமல்=அளவில்லாமல்
இசை+ஒளி=இசையொலி 
ஓய்வெடுத்தார்=ஓய்வு+எடுத்தார்      
மரம்+பொந்து=மரப்பொந்து 
```
