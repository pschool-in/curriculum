# கல்வி
```
எண்ணென்ப ஏனை எழுத்தென்ப இவ்விரண்டும்
கண்என்ப வாழும் உயிர்க்கு. 
```

எண் என்று சொல்லப்படுவன, எழுத்து என்று சொல்லப்படுவன ஆகிய இருவகைக் கலைகளையும் வாழும் மக்களுக்குக் கண்கள் என்று கூறுவர்.

```
கற்க கசடறக் கற்பவை கற்றபின்
நிற்க அதற்குத் தக. 
```
கற்கத் தகுந்த நூல்களைக் குற்றமறக் கற்க வேண்டும்; அவ்வாறு கற்ற பிறகு கற்ற கல்விக்குத் தக்கவாறு நெறியில் நிற்க வேண்டும்.

```
கேடில் விழுச்செல்வம் கல்வி யொருவற்கு
மாடல்ல மற்றை யவை
```
ஒருவனுக்கு செல்வங்களில் அழிவில்லாத செல்வமாக கல்வி அறிவு மட்டுமே இருக்கம். மற்ற செல்வங்கள் (நிலம், பணம், வீடு, பொருள், நகை) யாவும் இயற்கையினாலோ செயற்கையினாலோ நிலையில்லாது அழிய கூடியது ஆகும். ஆதலால் அவைகள் செல்வங்களே அல்ல. ஆதனால் கல்வியே சிறந்த செல்வமாகும்.


# புறங்கூறாமை

```
புறம்கூறிப் பொய்த்துயிர் வாழ்தலின் சாதல்
அறம்கூறும் ஆக்கம் தரும்.
```

புறங்கூறிப் பொய்யாக நடந்து உயிர்வாழ்தலைவிட, அவ்வாறு செய்யாமல் வறுமையுற்று இறந்துவிடுதல் அறநூல்கள் சொல்லும் ஆக்கத்தைத் தரும்.

# ஆள்வினையுடமை
```
தெய்வத்தான் ஆகா தெனினும் முயற்சிதன்
மெய்வருத்தக் கூலி தரும். 
```

ஊழின் காரணத்தால் ஒரு செயல் முடியாமல் போகுமாயினும், முயற்சி தன் உடம்பு வருந்திய வருத்தத்தின் கூலியையாவது கொடுக்கும்.


```
முயற்சி திருவினை ஆக்கும் முயற்றின்மை
இன்மை புகுத்தி விடும்.
```

விடாமுயற்சியானது வளத்தினை உண்டாக்கும்; முயற்சி செய்யாமல் இருப்பவனுக்கு வறுமைவந்து சேரும்.


# சுற்றந்தழால்

```
காக்கை கரவா கரைந்துண்ணும் ஆக்கமும்
அன்ன நீரார்க்கே உள.
```

காகம் தனக்குக் கிடைத்ததை மறைக்காமல் தன் சுற்றத்தாரைக் கூவி அழைத்து உண்ணும். அத்தகைய பண்பு உடையவர்களிடமே செல்வமும் சேரும்.


# மடியின்மை

```
மடியை மடியா ஒழுகல் குடியைக்
குடியாக வேண்டு பவர்.
```
தம் குடியைச் சிறப்புடைய குடியாகச் செய்ய விரும்புபவர், சோம்பலைத் துன்பமாகக் கருதி முயற்சியோடு வாழ்தல் வேண்டும்.



# தெரிந்து செயல்வகை

```
செய்தக்க அல்ல செயக்கெடும் செய்தக்க
செய்யாமை யானும் கெடும்.
```
செய்யத்தகாத செயல்களைச் செய்வதாலும் செய்யத்தக்க செயல்களைச் செய்யாமல் விடுவதாலும் தீமை உண்டாகும்.

```
எண்ணித் துணிக கருமம் துணிந்தபின்
எண்ணுவம் என்பது இழுக்கு 
```

செய்யத் தகுந்த செயலையும் வழிகளை எண்ணிய பிறகே துணிந்து தொடங்க வேண்டும். துணிந்தபின் எண்ணிப் பார்க்கலாம் என்பது குற்றமாகும்.

# பொறையுடைமை

```
அகழ்வாரைத் தாங்கும் நிலம்போலத் தன்னை 
இகழ்வார்ப் பொறுத்தல் தலை. 
```
தன்னையே தோண்டும்  மனிதனைக் கூட  கீழே  விழுந்து விடாமல் தாங்கும் நிலம் போல , பழிப்பவர்க்லயும்,தூற்றுபவர்களையும் கூட பொறுத்து கொள்வது தலையாய பண்பாகும்.

```
திறனல்ல தற்பிறர் செய்யினும் நோநொந்து
அறனல்ல செய்யாமை நன்று.
```
பிறர் தனக்குத் தரக்கூடாத துன்பத்தைக் தந்தாலும் மனம் நொந்து அறம் அல்லாத செயல்களைச் செய்யாமை சிறப்புக்குரியது.

```
மிகுதியான் மிக்கவை செய்தாரைத் தாம்தம்
தகுதியான் வென்று விடல்.
```
செருக்கினால் துன்பம் தந்தவரை நம்முடைய பொறுமையால் வெல்ல வேண்டும்.

# மக்கட்பேறு

```
குழலினிது யாழினிது என்பதம் மக்கள்
மழலைச்சொல் கேளா தவர்.
```

புல்லாங்குழல், யாழ் (வீணை போன்ற இசைக் கருவி ) வழியாக வரும் இசை  எல்லாம் இனியமாக இருக்கிறது என்று , தன் குழந்தைகளின் மழலை மொழி கேட்காதவர்  மட்டுமே கூறுவர்

# தீவினை அச்சம்
```
தீயவை தீய பயத்தலால் தீயவை
தீயினும் அஞ்சப் படும்.
```
தீயவை தீயவற்றையே தருதலால் தீயைவிடக் கொடியதாகக் கருதி அவற்றைச் செய்ய அஞ்சவேண்டும்.

```
மறந்தும் பிறன்கேடு சூழற்க சூழின்
அறஞ்சூழும் சூழ்ந்தவன் கேடு.
```
மறந்தும்கூடப் பிறருக்குக் கெடுதல் செய்ய நினைக்கக் கூடாது. நினைத்தால், நினைத்தவருக்குக் கெடுதல் செய்ய அறம் நினைக்கும்.


# அன்புடைமை

```
என்பி லதனை வெயில்போலக் காயுமே
அன்பி லதனை அறம்
```

எலும்பு இல்லாத உடம்போடு வாழும் புழுவை வெயில் காய்ந்து வருத்துவது போல் அன்பு இல்லாத உயிரை அறம் வருத்தும்.


# இறைமாட்சி

```
இயற்றலும் ஈட்டலுங் காத்தலும் காத்த
வகுத்தலும் வல்ல தரசு.
```

பொருள் வரும் வழிகளை மேன்மேலும் இயற்றலும் வந்த பொருள்களைச் சேர்த்தலும், காத்தலும் காத்தவற்றை வகுத்துச் செலவு செய்தலும் வல்லவன் அரசன்.

```
தூங்காமை கல்வி துணிவுடைமை இம்மூன்றும்
நீங்கா நிலனான் பவர்க்கு.
```

காலம் தாழ்த்தாத தன்மை, கல்வியுடைமை, துணிவுடைமை ஆகிய இந்த மூன்று பண்புகளும் நிலத்தை ஆளும் அரசனுக்கு நீங்காமல் இருக்க வேண்டியவை.



# வலியறிதல்

```
அளவறிந்து வாழாதான் வாழ்க்கை உளபோல
இல்லாகித் தோன்றாக் கெடும்.
```

பொருளின் அளவு அறிந்து வாழாதவனுடைய வாழ்க்கை பல வளமும் இருப்பது போல் தோன்றி இல்லாமல் மறைந்து கெட்டுவிடும்.

# கேள்வி

```
செல்வத்துள் செல்வம் செவிச்செல்வம் அச்செல்வம்
செல்வத்துள் எல்லாம் தலை
```

செவியால் கேட்டறியும் செல்வம், செல்வங்களுள் ஒன்றாகப் போற்றப்படும் செல்வமாகும்; அச் செல்வம் செல்வங்கள் எல்லாவற்றிலும் தலையானதாகும்.

```
எனைத்தானும் நல்லவை கேட்க அனைத்தானும்
ஆன்ற பெருமை தரும். 
```
எவ்வளவு சிறிதானாலும் நல்லவற்றைக் கேட்டால், கேட்ட அளவுக்குப் பெருமை உண்டாகும்.

```
நுணங்கிய கேள்வியர் அல்லார் வணங்கிய
வாயினர் ஆதல் அரிது.
```
நுட்பமான கேள்வியறிவு இல்லாதவர் அடக்கமான சொற்களைப் பேசுவது அரிது.

```
செவியிற் சுவையுணரா வாயுணர்வின் மாக்கள்
அவியினும் வாழினும் என்.
```
கேட்பதன் சுவையை உணராமல் நாவின் சுவை மட்டும் உணர்பவர் இறந்தால்தான் என்ன! இருந்தால்தான் என்ன!

# இடுக்கண் அழியாமை

```
இடும்பைக்கு இடும்பை படுப்பர் இடும்பைக்கு
இடும்பை படாஅ தவர். 
```

துன்பம் வந்தபோது அதற்காக வருந்திக் கலங்காதவர் அந்தத் துன்பத்திற்கே துன்பம் உண்டாக்கி அதை வென்று விடுவர்.


# நட்பு

```
செயற்கரிய யாவுள நட்பின் அதுபோல்
வினைக்கரிய யாவுள காப்பு. 
```
நட்பைப்போல் செய்துகொள்வதற்கு அருமையானவை எவை உள்ளன? அதுபோல் தொழிலுக்கு அரிய காவலாக இருப்பவை எவை உள்ளன?


# குற்றங்கடிதல்

```
வருமுன்னர்க் காவாதான் வாழ்க்கை எரிமுன்னர்
வைத்தூறு போலக் கெடும்.
```

பொருள்: பழிவருமுன்னே சிந்தித்து தம்மைக் காத்துக்கொள்ளாதவருடைய வாழ்க்கை, நெருப்பின் அருகில் வைக்கப்பட்ட வைக்கோல் போர் போல அழிந்துவிடும்.

```
தன்குற்றம் நீக்கிப் பிறர்குற்றம் காண்கிற்பின்
என்குற்றம் ஆகும் இறைக்கு.
```
தலைவன் முதலில் தன்குற்றத்தைக் கண்டு நீக்கி, அதன்பின் பிறருடைய குற்றத்தை ஆராய்ந்தால், அவனுக்கு எந்தப் பழியும் ஏற்படாது.


```
செருக்குஞ் சினமும் சிறுமையும் இல்லார்
பெருக்கம் பெருமித நீர்த்து.
```
செருக்கும் சினமும் காமமும் ஆகிய இந்தக் குற்றங்கள் இல்லாதவருடைய வாழ்வில் காணும் பெருக்கம் மேம்பாடு உடையதாகும்


# இடனறிதல்


```
தொடங்கற்க எவ்வினையும் எள்ளற்க முற்றும்
இடங்கண்ட பின்அல் லது.
```
பொருத்தமான இடத்தை அறியாமல் எந்தச் செயலை யும், தொடங்கவும் கூடாது; இகழவும் கூடாது.

```
கடல்ஓடா கால்வல் நெடுந்தேர் கடல்ஓடும்
நாவாயும் ஓடா நிலத்து.
```

வலிமையான சக்க ரங்களைக் கொண்ட பெரியதேர் கடலில் ஓட இயலாது.கடலில் ஓடும் கப்பல் தரை யில் ஓட இயலாது. அவரவர் தமக்குரியஇடங்களிலேயே சிறப்பா கச் செயல்பட முடியும்.


```
அஞ்சாமை அல்லால் துணைவேண்டா எஞ்சாமை
எண்ணி இடத்தால் செயின்.
```
செய்யும் வழிவகைகளைக் குறைவில்லாமல் எண்ணித் தக்க இடத்தில் பொருந்திச் செய்தால், அஞ்சாமை அல்லாமல் வேறு துணை வேண்டியதில்லை


# தெரிந்து வினையாடல்

```
நன்மையும் தீமையும் நாடி நலம்புரிந்த
தன்மையான் ஆளப் படும்.
```
நன்மையும் தீமையுமாகிய இரண்டையும் ஆராய்ந்து நன்மை தருகின்றவற்றையே விரும்புகின்ற இயல்புடையவன் (செயலுக்கு உரியவனாக) ஆளப்படுவான்.


# ஊக்கமுடைமை

```
ஆக்கம் அதர்வினாய்ச் செல்லும் அசைவிலா
ஊக்கம் உடையான் உழை.
```
சோர்வு இல்லாத ஊக்கம் உடையவனிடத்தில் ஆக்கமானது தானே அவன் உள்ள இடத்திற்கு வழி கேட்டுக்கொண்டு போய்ச் சேரும்.



# அழுக்காறாமை

```
ஒழுக்காறாக் கொள்க ஒருவன்த ன் நெஞ்சத்து
அழுக்காறு இலாத இயல்பு. 
```
ஒருவர் தன் நெஞ்சில் பொறாமையில்லாத குணத்தையே ஒழுக்க நெறியாகக் கொண்டு வாழ வேண்டும்.

```
அவ்விய நெஞ்சத்தான் ஆக்கமும் செவ்வியான்
கேடும் நினைக்கப் படும்.
```

பொறாமை கொண்டவருடைய செல்வமும், பொறாமை இல்லாதவருடைய வறுமையும் சான்றோரால் ஆராயப்படும்.



# புறங்கூறாமை

```
கண்நின்று கண்அறச் சொல்லினும் சொல்லற்க
முன்இன்று பின்நோக்காச் சொல்.
```

ஒருவருக்கு நேர்நின்று கடுமையான சொற்களைச் சொன்னாலும் சொல்லலாம். ஆனால், அவர் இல்லாதபோது புறங்கூறுதல் கூடாது.


```
ஏதிலார் குற்றம்போல் தம்குற்றம் காண்கிற்பின்
தீதுண்டோ மன்னும் உயிர்க்கு. 
```
பிறருடைய குற்றத்தைக் காண்பது போல், தன்னுடைய குற்றத்தையும் காண்பவருடைய வாழ்வில் துன்பம் இல்லை.



# அருளுடைமை 

```
அருட்செல்வம் செல்வத்துள் செல்வம் ப�ொருட்செல்வம்
பூரியார் கண்ணும் உள.
```
அருளாகிய செல்வமே செல்வங்களுள் சிறந்த செல்வமாகும். பொருட்செல்வம் இழிந்தவரிடத்திலும் உள்ளது.


```
வலியார்முன் தன்னை நினைக்க தான் தன்னின்
மெலியார்மேல் செல்லும் இடத்து.
```

ஒருவர் தன்னைவிட மெலிந்தவரை துன்புறுத்தும்போ து, தன்னை விட வலிமையுடையவரின் முன் தான் அஞ்சி நிற்கும் நிலையை எண்ணிப் பார்த்தல் வேண்டும்.


# வாய்மை

```
வாய்மை எனப்படுவது யாதெனின் யாதொன்றும்
தீமை இலாத சொலல்.
```

வாய்மை எனப்படுவது மற்றவர்க்கு ஒரு தீங்கும் தராத சொற்களைச் சொல்லுதல் ஆகும்.


```
தன்நெஞ்சு அறிவது பொய்யற்க பொய்த்தபின்   
தன்நெஞ்சே தன்னைச் சுடும்.
```

ஒருவர் தன் நெஞ்சறிய பொய் சொல்லக்கூடாது. அவ்வாறு கூறினால் அவர் நெஞ்சமே அவனை வருத்தும்.


```
உள்ளத்தால் பொய்யாது ஒழுகின் உலகத்தார்
உள்ளத்துள் எல்லாம் உளன். 
```
உள்ளத்தில் பொய் இல்லாமல் வாழ்பவர், உலகத்தார் உள்ளங்களில் எல்லாம் இருப்பவர் ஆவார்.


# வினைசெயல் வகை

```
�ொருள்கருவி காலம் வினைஇடன�ொடு ஐந்தும்
இருள்தீர எண்ணிச் செயல்.
```
வேண்டிய பொருள், ஏற்ற கருவி, தகுந்த காலம், மேற்கொள்ளும் செயலின் தன்மை , உரிய இடம் ஆகிய ஐந்தையும் ஐயம்தீர ஆராய்ந்து ஒரு செயலைச் செய்ய வேண்டும்.

```
வினையால் வினை யாக்கிக் கோடல் நனை கவுள்
யானையால் யானையாத் தற்று.
```

ஒரு யானையைக்கொண்டு மற்றொரு யானையைப் பிடிப்பர். அது போல ஒரு செயலைச் செய்யும் போதே அச்செயலால் மற்றொரு செயலையும் செய்து முடித்துக் கொள்ளல் வேண்டும்.


# அவை அஞ்சாமை

```
கற்றாருள் கற்றார் எனப்ப டுவர் கற்றார்முன்
கற்ற செலச்சொல்லு வார்.
```
தாம் கற்றவற்றைக் கற்றவர்முன் தெளிவாகச் சொல்ல வல்லவர், கற்றவருள் மிகவும் கற்றவராக மதிக்கப்படுவார்.

```
கற்றார்முன் கற்ற செலச்சொல்லித் தாம்கற்ற
மிக்காருள் மிக்க கொளல்.
```
கற்றவர் முன் தான் கற்றவற்றை மனத்தில் பதியும்படி சொல்லி, அவர்கள் கற்றவற்றையும் கேட்டு, அறிந்து கொள்ள வேண்டும்.

# நாடு

```
உறுபசியும் ஓவாப் பிணியும் செறுபகையும்
சேராது இயல்வது நாடு.
```

மிக்க பசியும், ஓயாத நோயும், அழிவு செய்யும் பகையும் சேராமல் நல்ல வகையில் நடைபெறுவதே நாடாகும்.


```
நாடென்ப நாடா வளத்தன நாடல்ல
நாட வளம்தரு நாடு.
```
பெரிய அளவில் முயற்சி இல்லாமல் வளம்த ரும் நாடே சிறந்த நாடாகும். முயற்சி செய்து சேரும் வளத்தை உடைய நாடு சிறந்த நாடு ஆகாது.
 

# அரண்

```
மணிநீரும் மண்ணும் மலையும் அணிநிழல்
காடும் உடையது அரண்.
```
தெளிந்த நீரும், நிலமும், மலையும், அழகிய நிழல் உடைய காடும் ஆகிய நான்கும் உள்ளதே அரண் ஆகும்.

```
எனைமாட்சித்து ஆகியக் கண்ணும் வினைமாட்சி
இல்லார்கண் இல்லது அரண்.
```
அரண் எவ்வளவு பெருமையுடை யதாக இருந்தாலும், செயல் சிறப்பு இல்லாதவரிடத்தில் அது பயனில்லாதது ஆகும்.
 
 
# பெருமை
```
பிறப்பொக்கும் எல்லா உயிர்க்கும் சிறப்புஒவ்வா
செய்தொழில் வேற்றுமை யான்.
```
பிறப்பால் மக்கள் அனைவரும் ஒத்த இயல்புடையவர்களே. அவர்கள் செய்யும் நன்மை, தீமையாகியச் செயல்களால் அவர்களது சிறப்பியல்புகள்
ஒத்திருப்பதில்லை.

```
பெருமை உடையவர் ஆற்றுவார் ஆற்றின்
அருமை உடைய செயல்.
```
உயர்ந்த பண்புகள ை உடையவர் செய்வதற்கு அரிய செயல்களை உரிய நெறிமுறையில் செய்து முடிப்பர்.
 
# நடுவுநிலைமை

```
தக்கார் தகவிலர் என்பது அவரவர்
எச்சத்தால் காணப் படும்.
```
நடுவு நிலைமை உடையவர், நடுவு நிலைமை இல்லாதவர் என்பது அவரவருக்குப் பின் எஞ்சி நிற்கும் புகழினாலும் பழியினாலும் அறியப்படும்.

```
சமன்செய்து சீர்தூக்கும் கோல்போல் அமைந்துஒருபால்
கோடாமை சான்றோர்க்கு அணி.
```
தான் சமமாக இருந்து தன்னிடம் வைக்கப்ப டும் பொருள்களின் எடையைத் துலாக்கோல் சரியாகக் காட்டும். அதுபோல நடுவுநிலைமையுடன் சரியாகச் செயல்படுவதே சான்றோர்க்கு அழகாகும்.



# கூடா ஒழுக்கம்
```
வலியில் நிலைமை யான் வல்லுருவம் பெற்றம்
புலியின்தோல் போர்த்துமேய்ந் தற்று.
```
மனத்தை அடக்கும் வல்லமை இல்லாதவர் மேற்கொண்ட வலிய தவக்கோலம், புலியின் தோலைப் போர்த்திக் கொண்ட பசு பயிரை மேய்ந்ததைப் போன்றது.

```
கணைகொடிது யாழ்கோடு செவ்விதுஆங்கு அன்ன
வினைபடு பாலால் கொளல்.
```
நேராக இருந்தாலும் அம்பு கொடியதாக இருக்கிறது. வளைவுடன் இருப்பினும் யாழின் கொம்பு இனிமையைத் தருகிறது. அதுபோல மக்களின் பண்புகளை அவரவர் தோற்றத்தால் அல்லாமல் செயல்வகையால் உணர்ந்துகொள்ள வேண்டும்.
 
# கல்லாமை
```
உளர்என்னும் மாத்திரையர் அல்லால் பயவாக்
களர்அனையர் கல்லா தவர்.
```
கல்லாதவர் பயனில்லாத களர்நிலம் போன்றவர். அவர் உயிரோடு இருக்கிறார் என்பதைத் தவிர அவரால் வேறு எந்தப் பயனும் இல்லை.

```
விலங்கொடு மக்கள் அனையர் இலங்குநூல்
கற்றாரோடு ஏனை யவர்.
```
கற்றவர்க்கும் கல்லாதவருக்கும் இடையே உள்ள வேறுபாடானது, மக்களுக்கும் விலங்குகளுக்கும் இடையே உள்ள வேறுபாட்டிற்கு இணையானது.


# தெரிந்து வினையாடல்
```
செய்வானை நாடி வினைந ாடிக் காலத்தோடு
எய்த உணர்ந்து செயல்.
```
செயலாற்றும் திறன் உடையவரையும் செய்யவேண் டிய செயலையும் செய்வதற்குரிய காலத்தையும் ஆராய்ந்து அச்செயலை நிறைவேற்ற வேண்டும்.

```
இதனை இதனால் இவன்முடிக்கும் என்றுஆய்ந்து
அதனை அவன்கண் விடல்.
```
இச்செயலை இந்தவகையால் இவர் செய் துமுடிப்பா ர் என்று ஆராய்ந்து அச்செயலை அவரிடம் ஒப்படைக்கவேண் டும்.

# செங்கோன்மை
```
ஓர்ந்துகண் ணோடாது இறைபுரிந்து யார்மாட்டும்
தேர்ந்துசெய் வஃதே முறை.
```
எதையும் நன்கு ஆராய்ந்து ஒருபக்கம் சா யாது நடுவுநிலையில் நின்று நடத்துவதே சிறந்த ஆட்சியாகும்.

```
இறைகாக்கும் வையகம் எல்லா ம் அவனை
முறைகாக்கும் முட்டாச் செயின்.
```

உலகத்து உயிர்களை எல்லாம் அரசர் காப்பாற்றுவார். அவரை அவரது குற்றமற்ற ஆட்சி காப்பாற்றும்.

# வெருவந்த செய்யாமை
```
தக்காங்கு நாடித் தலைசெல்லா வண்ணத்தால்
ஒத்தாங்கு ஒறுப்பது வேந்து.
```

ஒருவர் செய்த குற்ற த்தை முறையாக ஆராய்ந்து அவர் மீண்டும் குற்றம் செய்யாதவாறு தண் டிப்ப து அரசனின் கடமையாகும்.

```
இறைகடியன் என்றுஉரைக்கும் இன்னாச்சொல் வேந்தன்
உறைகடுகி ஒல்லைக் கெடும்.
```

நம் அரசர் கடுமையானவர் என்று குடிமக்களால் தூற்றப்படும் கொடுஞ்சொல்லை உடைய அரசர், தன் வாழ்நாள் குறைந்து விரைவில் அழிவார்.


# சொல்வன்மை

```
கேட்டார்ப் பிணிக்கும் தகையவாய்க் கேளாரும்
வேட்ப மொழிவதாம் சொல்.
```
கேட்பவரைத் தன்வ யப்ப டுத்துவதும் கேளாத வரைக் கேட்கத் தூண்டுவதும் சிறந்த சொல்லாற்ற லின் இயல்பாகும்.
```
சொல்லுக சொல்லைப் பிறிதுஓர்சொல் அச்சொல்லை
வெல்லும்சொல் இன்மை அறிந்து.
```
நாம் சொல்லும் சொல்லை வேறு சொல்லால் வெல்ல இயலாதவாறு சிறந்த சொற்களைத் தேர்ந்தெ டுத்துப் பேசவேண் டும்.

# அவையறிதல்
```
அவைஅறிந்து ஆராய்ந்து சொல்லுக சொல்லின்
தொகைஅறிந்த தூய்மை யவர்.
```
சொல்வள மும் நற்பண்பும் உடையவர்கள் தாம் பேசும் அவையின் தகுதி அறிந்து பேசுதல் வேண்டும்.

```
கற்றுஅறிந்தார் கல்வி விளங்கும் கசடறச்
சொல்தெரிதல் வல்லார் அகத்து.
```
சொற்களை ஆராயும் அறிஞர்நிறைந்த அவையில் பேசும்போதுதான் பலநூல்களைக் கற்றவரின் கல்வி பெருமையடையும்.


# படைச்செருக்கு
```
கான முயல்எய்த அம்பினில் யானை
பிழைத்தவேல் ஏந்தல் இனிது.
```
காட்டுமுயலை வீழ்த்திய அம்பினை ஏந்துவதைவிட யானைக்குக் குறிவைத்துத் தவறிய வேலை ஏந்துவது பெருமை தரும். (பெரிய முயற்சியே பெருமை தரும்.)

```
பேராண்மை என்ப தறுகண்ஒன்று உற்றக்கால்
ஊராண்மை மற்றுஅதன் எஃகு.
```
பகைவரை எதிர்த்து நிற்கும் வீரத்தை ஆண்மை என்று கூறுவர். பகைவருக்கும் துன்பம் வரும்போது, உதவி செய்தலை அந்த ஆண்மையின் கூர்மை என்று கூறுவர்.

# நட்பு

```
நவில்தொறும் நூல்நயம் போலும் பயில்தொறும்
பண்புடை யாளர் தொடர்பு.
```
நல்ல நூல்கள் படிக்கப் படிக்க இன்பம் தருவதுபோலப் பண்புடையவர் நட்பு பழகப் பழக இன்பம் தரும்.

```
நகுதல் பொருட்டுஅன்று நட்டல் மிகுதிக்கண்
மேற்சென்று இடித்தல் பொருட்டு.
```
நட்பு, சிரித்துப் பேசி மகிழ்வத ற்கு மட்டும் உரியது அன்று; நண்பர் தவறு செய்தால் அவரைக் கண்டித்துத் திருத்துவதற்கும் உரியது.

# நட்பு ஆராய்தல்
```
ஆய்ந்துஆய்ந்து கொள்ளாதான் கேண்மை கடைமுறை
தான்சாம் துயரம் தரும்.
```
மீண்டும் மீண்டும் ஆராய்ந்து பாராமல் ஒருவனுடன் கொண்ட நட்பு தா ம் சாகும் அளவுக்குத் துன்பம் தரும்.

```
கேட்டினும் உண்டுஓர் உறுதி கிளைஞரை
நீட்டி அளப்பதோர் கோல்.
```
நமக்கு வரும் துன்பத்தினாலும் ஒரு நன்மை உண்டு. அத்துன்பமே நமது நண்பர்களின் உண்மையான இயல்பை அளந்துகாட்டும் அளவுகோலாகும்.

# மானம்
```
பெருக்கத்து வேண்டும் பணிதல் சிறிய
சுருக்கத்து வேண்டும் உயர்வு.
```
செல்வம் மிகுந்த காலத்தில் பணிவுடன் நடந்துகொள்ள வேண்டும். வறுமை வந்த காலத்தில் ஊக்க ம் குறையாமல் பெருமித உணர்வுடன் வாழவேண்டும்.

```
குன்றின் அனையாரும் குன்றுவர் குன்றுவ
குன்றி அனைய செயின்.
```
மலையளவு பெருமை உடையவராக இருந்தா லும் குன்றிமணியளவு தவறு செய்தால் அவரது பெருமை அழிந்துவிடும்.

# பண்புடைமை
```
பண்புடையார்ப் பட்டுஉண்டு உலகம் அதுஇன்றேல்
மண்புக்கு மாய்வ து மன்.
```
பண்பு உடைய சான்றோ ரின் வழியில் நடப்பதால்தா ன் இவ்வுலகம் இன்னும் இயங்குகிறது; இல்லாவிட்டால் மண்ணுக்குள் புதைந்து அழிந்திருக்கும்.

```
பண்பிலான் பெற்ற பெருஞ்செல்வம் நன்பால்
கலம்தீமை யால்திரிந்து அற்று.
```
தூய்மையற்ற பாத்திரத்தில் வைக்கப்ப டும் பால் திரிந்துவிடும். அதுபோ ல நற்பண்பு இல்லாத வன் பெற்ற பெருஞ்செல்வ மும் யாருக்கும் பயனின்றி அழியும்.
