# கல்வி
```
எண்ணென்ப ஏனை எழுத்தென்ப இவ்விரண்டும்
கண்என்ப வாழும் உயிர்க்கு. 
```

எண் என்று சொல்லப்படுவன, எழுத்து என்று சொல்லப்படுவன ஆகிய இருவகைக் கலைகளையும் வாழும் மக்களுக்குக் கண்கள் என்று கூறுவர்.

```
கற்க கசடறக் கற்பவை கற்றபின்
நிற்க அதற்குத் தக. 
```
கற்கத் தகுந்த நூல்களைக் குற்றமறக் கற்க வேண்டும்; அவ்வாறு கற்ற பிறகு கற்ற கல்விக்குத் தக்கவாறு நெறியில் நிற்க வேண்டும்.

```
கேடில் விழுச்செல்வம் கல்வி யொருவற்கு
மாடல்ல மற்றை யவை
```
ஒருவனுக்கு செல்வங்களில் அழிவில்லாத செல்வமாக கல்வி அறிவு மட்டுமே இருக்கம். மற்ற செல்வங்கள் (நிலம், பணம், வீடு, பொருள், நகை) யாவும் இயற்கையினாலோ செயற்கையினாலோ நிலையில்லாது அழிய கூடியது ஆகும். ஆதலால் அவைகள் செல்வங்களே அல்ல. ஆதனால் கல்வியே சிறந்த செல்வமாகும்.


# புறங்கூறாமை

```
புறம்கூறிப் பொய்த்துயிர் வாழ்தலின் சாதல்
அறம்கூறும் ஆக்கம் தரும்.
```

புறங்கூறிப் பொய்யாக நடந்து உயிர்வாழ்தலைவிட, அவ்வாறு செய்யாமல் வறுமையுற்று இறந்துவிடுதல் அறநூல்கள் சொல்லும் ஆக்கத்தைத் தரும்.

# ஆள்வினையுடமை
```
தெய்வத்தான் ஆகா தெனினும் முயற்சிதன்
மெய்வருத்தக் கூலி தரும். 
```

ஊழின் காரணத்தால் ஒரு செயல் முடியாமல் போகுமாயினும், முயற்சி தன் உடம்பு வருந்திய வருத்தத்தின் கூலியையாவது கொடுக்கும்.


```
முயற்சி திருவினை ஆக்கும் முயற்றின்மை
இன்மை புகுத்தி விடும்.
```

விடாமுயற்சியானது வளத்தினை உண்டாக்கும்; முயற்சி செய்யாமல் இருப்பவனுக்கு வறுமைவந்து சேரும்.


# சுற்றந்தழால்

```
காக்கை கரவா கரைந்துண்ணும் ஆக்கமும்
அன்ன நீரார்க்கே உள.
```

காகம் தனக்குக் கிடைத்ததை மறைக்காமல் தன் சுற்றத்தாரைக் கூவி அழைத்து உண்ணும். அத்தகைய பண்பு உடையவர்களிடமே செல்வமும் சேரும்.


# மடியின்மை

```
மடியை மடியா ஒழுகல் குடியைக்
குடியாக வேண்டு பவர்.
```
தம் குடியைச் சிறப்புடைய குடியாகச் செய்ய விரும்புபவர், சோம்பலைத் துன்பமாகக் கருதி முயற்சியோடு வாழ்தல் வேண்டும்.



# தெரிந்து செயல்வகை

```
செய்தக்க அல்ல செயக்கெடும் செய்தக்க
செய்யாமை யானும் கெடும்.
```
செய்யத்தகாத செயல்களைச் செய்வதாலும் செய்யத்தக்க செயல்களைச் செய்யாமல் விடுவதாலும் தீமை உண்டாகும்.

```
எண்ணித் துணிக கருமம் துணிந்தபின்
எண்ணுவம் என்பது இழுக்கு 
```

செய்யத் தகுந்த செயலையும் வழிகளை எண்ணிய பிறகே துணிந்து தொடங்க வேண்டும். துணிந்தபின் எண்ணிப் பார்க்கலாம் என்பது குற்றமாகும்.

# பொறையுடைமை

```
அகழ்வாரைத் தாங்கும் நிலம்போலத் தன்னை 
இகழ்வார்ப் பொறுத்தல் தலை. 
```
தன்னையே தோண்டும்  மனிதனைக் கூட  கீழே  விழுந்து விடாமல் தாங்கும் நிலம் போல , பழிப்பவர்க்லயும்,தூற்றுபவர்களையும் கூட பொறுத்து கொள்வது தலையாய பண்பாகும்.


# மக்கட்பேறு

```
குழலினிது யாழினிது என்பதம் மக்கள்
மழலைச்சொல் கேளா தவர்.
```

புல்லாங்குழல், யாழ் (வீணை போன்ற இசைக் கருவி ) வழியாக வரும் இசை  எல்லாம் இனியமாக இருக்கிறது என்று , தன் குழந்தைகளின் மழலை மொழி கேட்காதவர்  மட்டுமே கூறுவர்


# அன்புடைமை

```
என்பி லதனை வெயில்போலக் காயுமே
அன்பி லதனை அறம்
```

எலும்பு இல்லாத உடம்போடு வாழும் புழுவை வெயில் காய்ந்து வருத்துவது போல் அன்பு இல்லாத உயிரை அறம் வருத்தும்.


# இறைமாட்சி

```
இயற்றலும் ஈட்டலுங் காத்தலும் காத்த
வகுத்தலும் வல்ல தரசு.
```

பொருள் வரும் வழிகளை மேன்மேலும் இயற்றலும் வந்த பொருள்களைச் சேர்த்தலும், காத்தலும் காத்தவற்றை வகுத்துச் செலவு செய்தலும் வல்லவன் அரசன்.

```
தூங்காமை கல்வி துணிவுடைமை இம்மூன்றும்
நீங்கா நிலனான் பவர்க்கு.
```

காலம் தாழ்த்தாத தன்மை, கல்வியுடைமை, துணிவுடைமை ஆகிய இந்த மூன்று பண்புகளும் நிலத்தை ஆளும் அரசனுக்கு நீங்காமல் இருக்க வேண்டியவை.



# வலியறிதல்

```
அளவறிந்து வாழாதான் வாழ்க்கை உளபோல
இல்லாகித் தோன்றாக் கெடும்.
```

பொருளின் அளவு அறிந்து வாழாதவனுடைய வாழ்க்கை பல வளமும் இருப்பது போல் தோன்றி இல்லாமல் மறைந்து கெட்டுவிடும்.

# கேள்வி

```
செல்வத்துள் செல்வம் செவிச்செல்வம் அச்செல்வம்
செல்வத்துள் எல்லாம் தலை
```

செவியால் கேட்டறியும் செல்வம், செல்வங்களுள் ஒன்றாகப் போற்றப்படும் செல்வமாகும்; அச் செல்வம் செல்வங்கள் எல்லாவற்றிலும் தலையானதாகும்.


# இடுக்கண் அழியாமை

```
இடும்பைக்கு இடும்பை படுப்பர் இடும்பைக்கு
இடும்பை படாஅ தவர். 
```

துன்பம் வந்தபோது அதற்காக வருந்திக் கலங்காதவர் அந்தத் துன்பத்திற்கே துன்பம் உண்டாக்கி அதை வென்று விடுவர்.


# நட்பு

```
செயற்கரிய யாவுள நட்பின் அதுபோல்
வினைக்கரிய யாவுள காப்பு. 
```
நட்பைப்போல் செய்துகொள்வதற்கு அருமையானவை எவை உள்ளன? அதுபோல் தொழிலுக்கு அரிய காவலாக இருப்பவை எவை உள்ளன?


# குற்றங்கடிதல்

```
செருக்குஞ் சினமும் சிறுமையும் இல்லார்
பெருக்கம் பெருமித நீர்த்து.
```
செருக்கும் சினமும் காமமும் ஆகிய இந்தக் குற்றங்கள் இல்லாதவருடைய வாழ்வில் காணும் பெருக்கம் மேம்பாடு உடையதாகும்


# இடனறிதல்

```
அஞ்சாமை அல்லால் துணைவேண்டா எஞ்சாமை
எண்ணி இடத்தால் செயின்.
```
செய்யும் வழிவகைகளைக் குறைவில்லாமல் எண்ணித் தக்க இடத்தில் பொருந்திச் செய்தால், அஞ்சாமை அல்லாமல் வேறு துணை வேண்டியதில்லை


# தெரிந்து வினையாடல்

```
நன்மையும் தீமையும் நாடி நலம்புரிந்த
தன்மையான் ஆளப் படும்.
```
நன்மையும் தீமையுமாகிய இரண்டையும் ஆராய்ந்து நன்மை தருகின்றவற்றையே விரும்புகின்ற இயல்புடையவன் (செயலுக்கு உரியவனாக) ஆளப்படுவான்.


# ஊக்கமுடைமை

```
ஆக்கம் அதர்வினாய்ச் செல்லும் அசைவிலா
ஊக்கம் உடையான் உழை.
```
சோர்வு இல்லாத ஊக்கம் உடையவனிடத்தில் ஆக்கமானது தானே அவன் உள்ள இடத்திற்கு வழி கேட்டுக்கொண்டு போய்ச் சேரும்.



# அழுக்காறாமை

```
ஒழுக்காறாக் கொள்க ஒருவன்த ன் நெஞ்சத்து
அழுக்காறு இலாத இயல்பு. 
```
ஒருவர் தன் நெஞ்சில் பொறாமையில்லாத குணத்தையே ஒழுக்க நெறியாகக் கொண்டு வாழ வேண்டும்.

```
அவ்விய நெஞ்சத்தான் ஆக்கமும் செவ்வியான்
கேடும் நினைக்கப் படும்.
```

பொறாமை கொண்டவருடைய செல்வமும், பொறாமை இல்லாதவருடைய வறுமையும் சான்றோரால் ஆராயப்படும்.



# புறங்கூறாமை

```
கண்நின்று கண்அறச் சொல்லினும் சொல்லற்க
முன்இன்று பின்நோக்காச் சொல்.
```

ஒருவருக்கு நேர்நின்று கடுமையான சொற்களைச் சொன்னாலும் சொல்லலாம். ஆனால், அவர் இல்லாதபோது புறங்கூறுதல் கூடாது.


```
ஏதிலார் குற்றம்போல் தம்குற்றம் காண்கிற்பின்
தீதுண்டோ மன்னும் உயிர்க்கு. 
```
பிறருடைய குற்றத்தைக் காண்பது போல், தன்னுடைய குற்றத்தையும் காண்பவருடைய வாழ்வில் துன்பம் இல்லை.



# அருளுடைமை 

```
அருட்செல்வம் செல்வத்துள் செல்வம் ப�ொருட்செல்வம்
பூரியார் கண்ணும் உள.
```
அருளாகிய செல்வமே செல்வங்களுள் சிறந்த செல்வமாகும். பொருட்செல்வம் இழிந்தவரிடத்திலும் உள்ளது.


```
வலியார்முன் தன்னை நினைக்க தான் தன்னின்
மெலியார்மேல் செல்லும் இடத்து.
```

ஒருவர் தன்னைவிட மெலிந்தவரை துன்புறுத்தும்போ து, தன்னை விட வலிமையுடையவரின் முன் தான் அஞ்சி நிற்கும் நிலையை எண்ணிப் பார்த்தல் வேண்டும்.


# வாய்மை

```
வாய்மை எனப்படுவது யாதெனின் யாதொன்றும்
தீமை இலாத சொலல்.
```

வாய்மை எனப்படுவது மற்றவர்க்கு ஒரு தீங்கும் தராத சொற்களைச் சொல்லுதல் ஆகும்.


```
தன்நெஞ்சு அறிவது பொய்யற்க பொய்த்தபின்   
தன்நெஞ்சே தன்னைச் சுடும்.
```

ஒருவர் தன் நெஞ்சறிய பொய் சொல்லக்கூடாது. அவ்வாறு கூறினால் அவர் நெஞ்சமே அவனை வருத்தும்.


```
உள்ளத்தால் பொய்யாது ஒழுகின் உலகத்தார்
உள்ளத்துள் எல்லாம் உளன். 
```
உள்ளத்தில் பொய் இல்லாமல் வாழ்பவர், உலகத்தார் உள்ளங்களில் எல்லாம் இருப்பவர் ஆவார்.


# வினைசெயல் வகை

```
�ொருள்கருவி காலம் வினைஇடன�ொடு ஐந்தும்
இருள்தீர எண்ணிச் செயல்.
```
வேண்டிய பொருள், ஏற்ற கருவி, தகுந்த காலம், மேற்கொள்ளும் செயலின் தன்மை , உரிய இடம் ஆகிய ஐந்தையும் ஐயம்தீர ஆராய்ந்து ஒரு செயலைச் செய்ய வேண்டும்.

```
வினையால் வினை யாக்கிக் கோடல் நனை கவுள்
யானையால் யானையாத் தற்று.
```

ஒரு யானையைக்கொண்டு மற்றொரு யானையைப் பிடிப்பர். அது போல ஒரு செயலைச் செய்யும் போதே அச்செயலால் மற்றொரு செயலையும் செய்து முடித்துக் கொள்ளல் வேண்டும்.


# அவை அஞ்சாமை

```
கற்றாருள் கற்றார் எனப்ப டுவர் கற்றார்முன்
கற்ற செலச்சொல்லு வார்.
```
தாம் கற்றவற்றைக் கற்றவர்முன் தெளிவாகச் சொல்ல வல்லவர், கற்றவருள் மிகவும் கற்றவராக மதிக்கப்படுவார்.

```
கற்றார்முன் கற்ற செலச்சொல்லித் தாம்கற்ற
மிக்காருள் மிக்க கொளல்.
```
கற்றவர் முன் தான் கற்றவற்றை மனத்தில் பதியும்படி சொல்லி, அவர்கள் கற்றவற்றையும் கேட்டு, அறிந்து கொள்ள வேண்டும்.

# நாடு

```
உறுபசியும் ஓவாப் பிணியும் செறுபகையும்
சேராது இயல்வது நாடு.
```

மிக்க பசியும், ஓயாத நோயும், அழிவு செய்யும் பகையும் சேராமல் நல்ல வகையில் நடைபெறுவதே நாடாகும்.


```
நாடென்ப நாடா வளத்தன நாடல்ல
நாட வளம்தரு நாடு.
```
பெரிய அளவில் முயற்சி இல்லாமல் வளம்த ரும் நாடே சிறந்த நாடாகும். முயற்சி செய்து சேரும் வளத்தை உடைய நாடு சிறந்த நாடு ஆகாது.
 

# அரண்

```
மணிநீரும் மண்ணும் மலையும் அணிநிழல்
காடும் உடையது அரண்.
```
தெளிந்த நீரும், நிலமும், மலையும், அழகிய நிழல் உடைய காடும் ஆகிய நான்கும் உள்ளதே அரண் ஆகும்.

```
எனைமாட்சித்து ஆகியக் கண்ணும் வினைமாட்சி
இல்லார்கண் இல்லது அரண்.
```
அரண் எவ்வளவு பெருமையுடை யதாக இருந்தாலும், செயல் சிறப்பு இல்லாதவரிடத்தில் அது பயனில்லாதது ஆகும்.
